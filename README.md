# 📊 FinSight - Financial Analytics Dashboard (Power BI)

## Project Overview

FinSight is a Power BI financial analytics dashboard developed to analyze customer behavior, transaction patterns, fraud activities, and overall financial performance.

The dashboard helps financial institutions gain actionable insights from customer and transaction data through interactive visualizations and KPI tracking.


## Business Problem

Financial institutions generate large volumes of customer and transaction data every day. Without proper analytics, identifying customer trends, monitoring fraud risks, and understanding transaction performance becomes difficult.

This dashboard addresses:

- Customer segmentation analysis
- Transaction performance monitoring
- Fraud detection insights
- Revenue and fee analysis
- Customer demographic trends
- Channel-wise transaction behavior

# KPI Questions Answered

### Customer Analysis

1. How many customers are registered?

2. Which customer segment contributes the most?

3. What is the gender distribution of customers?

4. Which states have the highest customer count?

5. Which occupations dominate the customer base?

6. What is the average annual income of customers?


### Transaction Analysis

7. What is the total transaction volume?

8. What is the total transaction value?

9. Which transaction type is most frequently used?

10. Which channel generates the highest transactions?

11. What percentage of transactions are successful?

12. Which merchant categories contribute most to transaction value?


### Fraud Analysis

13. How many fraudulent transactions were detected?

14. What is the fraud rate?

15. Which channels have higher fraud occurrences?

16. What are the average risk scores across transactions?

17. Which transaction categories are most vulnerable to fraud?


### Financial Analysis

18. How much fee revenue is generated?

19. How much tax revenue is collected?

20. What is the net transaction amount?

21. Which customer segment contributes most to revenue?
